Q&A's - Especially for Waiting Parents

By Bonnie Wong

Question: I see there are a number of situations in which files in theory can be expired. Can you comment?

Answer: From CCAA's prospective, once a file has been registered by CCAA and the police checks and medical reports on the applicant(s) are dated within a year prior to the registration date, and no major changes happened to the pplicant(s) while the file is being processed by CCAA, the file will be theoritically valid until CCAA issues the travel approval (refered to Notice of Coming) to the applicant (s). A file is required to be updated by CCAA under (but not limited to) the following situations:

  • the file has never been registered by CCAA for many months due to whatever reason
  • the applicant(s) has/have requested CCAA to put their file on hold for some time and decide to reactivate the file
  • there has been major changes happened to the applicant(s), such as a new child added to the family, change of marital status, deterioration of health etc.

Please note BC regulations may have different requirements for a file to be updated. For further clarification/details on the above, please contact:
